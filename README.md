

14 de febrero de 2025. Desde el Virgen de Gracia vamos a lanzar un bot en telegram para que encuentre tu pareja perfecta. Para ello deben hablar a nuestro bot lanzado por la escuela para encontrar el amor en caso de que no haya visitado ya tu puerta cupido.
Instrucciones de uso. A continuación, listamos los comandos disponibles para que el usuario:
-	/start Te da la bienvenida a la aplicación, con un mensaje personalizado.
-	/love Comienza el cuestionario para que finalmente te de las 3 personas más afines.
-	/stop Paro el cuestionario porque quiero volver a contestar las preguntas.
-	/back Quiero volver a contestar la pregunta anterior porque me he equivocado.

Requisitos para el programador:
-	Se proporciona la lista de usuarios para analizar los datos e ir elaborando las preguntas.
-	Queda a elección del programador la afinidad del algoritmo, se valorará MUY positivamente que salgan resultados coherentes.
-	Desplegar el bot en Render o Railways.
Entrega:
Git utilizado para el despliegue del bot.
Documento de evidencias con 3 casos diferentes mostrando el resultado esperado.
